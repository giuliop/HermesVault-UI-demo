# Hermes Vault UI Demo

This is a demo user interface created to play around with and gather feedback.

It does not interact with the blockchain and the only wallet operations are connect/disconnect and signing messages, no transaction is ever signed.

Let us know what you think, any suggestion, feedbacl and criticism more than welcome !

Access the demo at: https://giuliopizzini.com/hermesvault-ui-demo/

You can read the Hermes Vault whitepaper [here](https://github.com/giuliop/HermesVault)

### How to use

You can make (fake) deposits and withdrawals.

#### Deposit

1. Connect your wallet (only Pera is supported for the demo)
2. Make a deposit of any amount, you don't need to have the actual algo in your account
3. Sign with Pera, as you'll see you are not signing a transaction, just a message.
4. Remember to save the secret note to withdraw later !

#### Withdraw

1. Select an arbitrary Algorand address as recipient (no real algo will be sent)
2. Select an amount to withdraw below what you deposited
3. Paste in the secret note from your deposit
4. Remember to save the new secret note to withdraw any balance left !
